# Business Performance & Customer Engagement Analysis

## Project Overview
This project analyzes customer orders and engagement data for an ed-tech business using SQL and Tableau.

## Objectives
- Analyze monthly revenue trends
- Compare acquisition channel performance
- Study relationship between customer engagement and revenue

## Tools Used
- MySQL
- Tableau

## Dashboard Insights
- Organic channel generated highest revenue
- Revenue peaked in February
- Customers with higher engagement showed higher spending

## Files
- data/ : raw dataset
- sql/ : analysis queries
- dashboard/ : Tableau workbook & screenshots
